# WebDoctor

## About

A stackoverflow like Q&A site implemented in Python. It was a teamwork project in the second module of my Codecool studies where I started to learn how to build full-stack web applications.

## Technologies

- Python 3.8.2
- Flask
- Jinja templating engine
- PostgreSQL
- Bcrypt
- HTML & CSS
